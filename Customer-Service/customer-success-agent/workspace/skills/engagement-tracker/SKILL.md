# Engagement Tracker Skill

Monitor product usage and engagement.

## Key Metrics

| Metric | Healthy Range |
|--------|---------------|
| Logins/week | > 3 |
| Features used | > 50% |
| Sessions/month | > 20 |
| Time in app | > 2 hrs/week |

## Trending

- Compare to baseline
- Track month-over-month
- Identify declining usage
- Spot engagement issues

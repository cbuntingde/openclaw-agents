# NPS Monitor Skill

Track and analyze Net Promoter Score.

## NPS Ranges

| Score | Category |
|-------|----------|
| 9-10 | Promoter |
| 7-8 | Passive |
| 0-6 | Detractor |

## NPS Calculation

```
NPS = % Promoters - % Detractors
```

## Tracking

- Collect after key interactions
- Track monthly trends
- Segment by customer tier
- Analyze feedback themes

# Dunning Manager

Manages dunning sequences and communications.

## Capabilities

- Execute dunning letter sequences
- Customize communications by aging
- Track dunning effectiveness
- Escalate when needed

## Dunning Stages

- Stage 1: Friendly reminder (1-15 days)
- Stage 2: Urgent payment request (16-30 days)
- Stage 3: Final notice (31-45 days)
- Stage 4: Collections escalation (45+ days)

## Output Format

- Dunning queue by stage
- Effectiveness metrics
- Escalation needs

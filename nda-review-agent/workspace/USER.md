# USER.md - NDA Review Agent

## User Profile

- **User**: Legal team, sales, business development, HR
- **Organization**: Any organization handling confidential information
- **Experience Level**: Business professionals

## Primary Use Cases

- Review incoming NDAs for concerning terms
- Track NDA status and signatures
- Monitor expiration dates and renewals
- Maintain NDA repository awareness
- Flag overdue renewals or expirations

## Preferences

- Clear deadline alerts
- Summary of key NDA terms
- Status tracking for all parties
- Renewal reminders well in advance

# TOOLS.md - SEO Agent

## Workspace Tools

This file contains notes about local tools and conventions.

## Available Skills

### keyword-tracker
- Purpose: Track keyword rankings
- Use for: Daily ranking checks, position monitoring
- Key functions: Query ranking APIs, track changes, alert on drops

### content-optimizer
- Purpose: Optimize content for SEO
- Use for: Analyzing existing content, recommendations
- Key functions: Analyze keywords, density, readability, structure

### ranking-monitor
- Purpose: Monitor overall SEO performance
- Use for: Domain authority, backlinks, technical health
- Key functions: Track DA, monitor backlinks, check site health

## Conventions

- Always cite data sources
- Focus on actionable recommendations
- Set realistic expectations
- Document ranking changes
- Track progress over time

# Feedback Collector Skill

Gather onboarding feedback at key milestones.

## Feedback Points

| Day | Type | Questions |
|-----|------|-----------|
| Day 7 | Week 1 | How's it going? Any blockers? |
| Day 30 | 30-day | How was onboarding? Suggestions? |
| Day 90 | 90-day | Full review of experience |

## Feedback Template

```
## Onboarding Feedback

**Employee**: [NAME]
**Review Period**: Day [X]
**Date**: [DATE]

### Questions
1. How would you rate your onboarding experience?
2. Were you set up with everything you needed?
3. Any suggestions for improvement?

### Responses
[ANSWERS]

### Overall Rating: [1-5]
```

## Notes

- Make feedback collection easy
- Act on feedback received
- Track trends over time

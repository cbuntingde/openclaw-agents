# Exception Handler Skill

Handle payroll exceptions and process adjustments.

## Common Exceptions

- Missing time entries
- Rate discrepancies
- Deduction errors
- Manual checks
- Reversals

## Process

1. Identify exception
2. Research cause
3. Calculate adjustment
4. Get approval
5. Process correction

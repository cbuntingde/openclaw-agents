# MEMORY.md - Access Management Agent

## Review Cycles

- **Quarterly Access Reviews**: Complete by end of each quarter
- **Annual Privileged Access Review**: Complete by year-end
- **Monthly Access Certification**: New access and exceptions

## Access Policies

### Least Privilege Principle
- Users receive minimum access required for job function
- Elevated access requires documented business justification
- All elevated access undergoes quarterly review

### Access Expiration
- Temporary access expires automatically
- Project-based access expires on project completion
- Contractor access reviewed monthly

### Deprovisioning SLA
- Termination: Same day disable
- Offboarding: Within 24 hours
- Role change: Within 48 hours

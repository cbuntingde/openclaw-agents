# Server Monitor Agent

[![OpenClaw v2026.1+](https://img.shields.io/badge/OpenClaw-v2026.1+-blue.svg)](https://docs.openclaw.ai)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

A professional OpenClaw agent for server monitoring. Monitors infrastructure health, performance, and capacity.

## Features

- **Health Checks** — Monitor server health status
- **Performance Monitoring** — Track performance metrics
- **Capacity Planning** — Plan infrastructure capacity

## Skills

- `health-check` - Check server health
- `performance-monitor` - Monitor performance
- `capacity-planner` - Plan capacity

# TOOLS.md - Treasury Agent

## Workspace Tools

This file contains notes about local tools and conventions.

## Available Skills

### cashflow-forecaster
- Purpose: Forecast cash flow patterns
- Use for: Predicting future cash positions, scenario modeling
- Key functions: Analyze trends, build forecasts, model scenarios

### liquidity-monitor
- Purpose: Monitor liquidity positions
- Use for: Daily cash position, working capital tracking
- Key functions: Track balances, monitor ratios, flag issues

### investment-tracker
- Purpose: Track investments
- Use for: Investment portfolio monitoring, performance reporting
- Key functions: Monitor returns, track allocations, rebalance

## Conventions

- Always use exact numbers
- Include source citations
- Flag uncertainties clearly
- Recommend specific actions
- Maintain confidentiality

# Budget Forecast Agent

[![OpenClaw v2026.1+](https://img.shields.io/badge/OpenClaw-v2026.1+-blue.svg)](https://docs.openclaw.ai)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

A professional OpenClaw agent for budget forecasting. Analyzes spending patterns, predicts burn rate, and alerts on budget thresholds.

## Skills

- `spending-analyzer` - Analyze spending patterns
- `burn-forecaster` - Predict burn rate
- `threshold-alerter` - Alert on budget breaches
